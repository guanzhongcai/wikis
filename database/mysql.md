# SQLite

SQLite 网站 ( https://www.sqlite.org/ ) 本身使用 SQLite，截至2015年时，它每天处理大约 400K 到 500K 的 HTTP 请求，其中大约 15-20% 是动态的接触数据库的页面。动态内容[每个网页](https://www.sqlite.org/np1queryprob.html)使用[大约 200 条 SQL 语句](https://www.sqlite.org/np1queryprob.html)。此设置在单个 VM 上运行，该 VM 与其他 23 台共享物理服务器，但在大多数情况下仍将平均负载保持在 0.1 以下。
